# Claude Code Tutorial #3 - Context (128kbit_AAC-English).mp3

## Overview

This tutorial focuses on managing context within a Claude code session. The use of context files, the '@' symbol, and various commands help in improving response accuracy by providing additional relevant information to prompts.

## Main Ideas

1. Manually adding files as context using the '@' symbol followed by a path.
2. Selecting a portion of code within a file for specific focus as context.
3. Adding images as context by dragging and dropping them into the chat session.
4. Clearing the entire session history and context with the 'clear' command.
5. Compacting the session history into a small summary using the '/compact' command.
6. Rewinding to a previous point in the session and removing all later chats and files from the context by tapping escape twice.
7. Exiting, resuming, and managing the conversation history with the "/exit", "/resume", and "clear conversation history" commands.
8. Understanding the Context Window concept and ways to reduce context bloat.

## Key Manipulations and Scripting or Coding Used

- Using the '@' symbol for adding files and referencing specific files as context.
- Selecting a portion of code within a file.
- Dragging and dropping images into the chat session.
- Utilizing the 'clear', '/compact', '/exit', "/resume", and "clear conversation history" commands.

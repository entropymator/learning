# Claude Code Tutorial #9 - Claude Code with GitHub (128kbit_AAC-English)

This tutorial focuses on utilizing the Cloud Code GitHub App to automate tasks and integrate Claude, an automated tool, for fixing issues in a GitHub repository.

## Main Ideas

1. Setting up the Cloud Code GitHub App within the Cloud Code chat session
2. Using Claude to fix issues in a GitHub repository
3. Automated actions such as auto-reviewing pull requests and working on issues on new feature branches
4. Previewing changes after merging pull requests
5. Exploring action files for additional integrations of Claude within different workflows

## Key Manipulations and Scripting/Coding Used

1. Installing the GitHub CLI
2. Using a built-in slash command in Cloud Code (`/install --github-app`) to set up the Cloud Code GitHub App
3. Authorizing the app on GitHub, choosing the repository, and creating a long-live token with an Anthropic subscription
4. Creating an issue, mentioning Claude, watching as Claude works on the issue, committing, pushing changes, and providing a pull request creation link
5. Previewing changes in a browser after merging the pull request
6. Exploring action files for additional integrations of Claude within different workflows

## Reflective/Quiz-Style Questions

1. What are the benefits of using the Cloud Code GitHub App to automate tasks?
2. How does Claude help in fixing issues in a GitHub repository?
3. Can you describe the process of previewing changes after merging a pull request created by Claude?

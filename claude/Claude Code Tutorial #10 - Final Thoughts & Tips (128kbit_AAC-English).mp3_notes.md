# Claude Code Tutorial #10 - Final Thoughts & Tips (128kbit_AAC-English).mp3

In this tutorial, we delve into best practices and tips when working with the Claude AI code assistant.

## Main Ideas
- Verify AI-generated code for accuracy to foster a better understanding of the codebase.
- Effectively utilize context to keep the AI focused and concise.
- Clear AI context or start a new session when dealing with new features or tasks.
- Use AI as a coding assistant for learning and skill improvement, rather than relying on it solely for problem-solving.
- Leverage shortcuts within the AI system to navigate through previous prompts, clear chat history, enter bash mode, and navigate back in the chat history.

## Key Manipulations and Scripting or Coding used
- Utilize arrow keys to cycle through previous prompts.
- Escape out of commands or tools within the AI system.
- Enter bash mode using the question mark (?).
- Double escape (\) to navigate back in the chat history.

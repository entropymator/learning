# Claude Code Tutorial #1 - Introduction & Setup (128kbit_AAC-English).mp3

This tutorial introduces you to Claude Code, an AI-powered coding tool by Anthropic that lives directly in the terminal and integrates with your existing development workflows. Unlike traditional IDEs, Claude understands the codebase of various projects and generates more tailored and appropriate code on a project-by-project basis.

## Main Ideas
1. Introduction to Claude Code
    - AI-powered agentic coding tool by Anthropic
    - Integrates with existing development workflows without requiring a specific IDE
    - Understands the codebase of various projects and generates more tailored and appropriate code
    - Can be used within GitHub workflows for automating code reviews on pull requests

2. Usage and Installation
    - Install Cloud Code via npm command or directly in a Windows terminal (no longer requires WSL)
    - Sign up for an account and choose a pricing plan (Pro: $17/month; Max plans available)

3. Setup and Usage
    - Navigate to the project you wish to work on using the terminal
    - Type "Claude" to initiate interaction with Cloud Code

4. Key Manipulations and Scripting or Coding Used
   - npm install hyphen g @anthropic/claude-code (for installation)
   - `/terminal setup` (to install shift plus enter key binding for new lines in the chat window with cloud code)
   - Various Git commands (e.g., switch branches, merge branches, resolve conflicts, look up commits or branches)
